# English #
## How to build ##

### Dependencies ###

Library    | Version | Reason
---	   | ---     | ---
libxml2	   | >= 2.9  | Parsing of the configuration file.
SDL2	   | >= 2.0  | Display everything.
SDL2_Image | >= 2.0  | PNG images.

### Compile ###

Just type ``make`` at the root of the project!
